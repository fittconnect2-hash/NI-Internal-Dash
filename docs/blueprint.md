# **App Name**: DineNet Admin Dashboard

## Core Features:

- Sidebar Navigation: Provide a fixed left sidebar for intuitive navigation to different sections such as Dashboard, Tenants, Outlets, Users, Gateways, Credentials, and Merchants.
- Tenant Overview Display: Render a main content area that dynamically displays tenant profiles in a customizable grid or list view, showcasing essential details like name, contact information, and current configuration status.
- Tenant Search & Filtering: Implement a functional search bar allowing users to find tenants by name, email, or phone number, complemented by a filter control to refine results based on various statuses.
- Tenant Addition & Editing: Enable users to seamlessly add new tenant records and modify existing tenant details directly through interactive cards and dedicated forms.
- Tenant Data Management: Establish a robust backend system for the secure storage, retrieval, and management of all tenant, user, outlet, and payment gateway information necessary for the dashboard's operations.
- Tenant Status Assistant Tool: An AI tool that intelligently analyzes tenant data and configurations to automatically identify or suggest potential issues and outstanding tasks, such as 'Configuration pending' statuses, improving management efficiency.

## Style Guidelines:

- Primary color: A vibrant and professional blue, represented as RGB(40, 114, 255) / #2872FF, used for main interactive elements, active navigation states, and calls to action.
- Background color: A very light, subtly bluish-grey, represented as RGB(243, 247, 252) / #F3F7FC, providing a clean, open, and modern base for the dashboard layout.
- Accent color: A distinct turquoise, represented as RGB(52, 187, 218) / #34BBDA, providing contrast for secondary highlights or interactive elements while maintaining visual harmony.
- Semantic Color - Pending Status: A soft yellow background, RGB(255, 248, 219) / #FFF8DB, with muted yellow text, RGB(162, 147, 0) / #A29300, specifically used to highlight 'Configuration pending' statuses for immediate user attention.
- Body and headline font: 'Inter' (sans-serif) is recommended for its modern, neutral, and highly legible characteristics, ensuring clarity across all textual content.
- Use a consistent set of minimalist, line-art style icons for all navigational elements and interactive features, aligning with the clean and professional aesthetic of the dashboard.
- Implement a two-column layout featuring a fixed-width left navigation sidebar and a flexible main content area, designed to adapt for responsive display of tenant cards in both grid and list formats.
- Incorporate subtle hover effects for navigational links and buttons, along with smooth transitions for content updates, to enhance user interaction and provide a polished experience.
